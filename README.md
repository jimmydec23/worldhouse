# worldhouse

## development env
* node: v10.24.1
* truffle
* Metamask browser extension

## development
* start dev server:
  ```
  yarn dev
  ```
* start dev client:
  ```
  yarn cdev
  ```
* open webpage: http://localhost

## build 
* build frontend app
```
yarn cbuild
```
* start server
```
yarn start
```
* open webpage: http://localhost
